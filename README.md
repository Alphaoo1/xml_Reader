# xml_Reader_test_project
This project is to test out xml reader using c++ and the BOOST library.

## Note

To build this project 

```
mkdir build
cd build
cmake ../
make
```

or use

```
cmake ../ && make && ./trial
```

## Usage 

```
./xml_Reader_test_project /path/to/xml/file/filename.xml_Reader_test_project
```
